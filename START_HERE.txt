════════════════════════════════════════════════════════════════════════════════
 HOW EMAIL WORKS
════════════════════════════════════════════════════════════════════════════════

User fills contact form
         ↓
Clicks "Send Message"
         ↓
Form data sent to /api/contact (your API route)
         ↓
API calls Resend with user's message
         ↓
Resend sends email to awaiss.dev@gmail.com
         ↓
Email arrives in your inbox ✅
         ↓
Success message shown to user

All automatic. No backend server needed.

 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

❌ Email not working?
  → Check Resend API key is correct
  → Verify it's in Vercel environment variables
  → Redeploy Vercel after adding key

❌ Site won't load?
  → Check GitHub push succeeded
  → Check build logs in Vercel dashboard
  → Try redeploying

❌ Need to update something?
  → Edit files locally
  → git add . && git commit -m "update" && git push
  → Vercel auto-redeploys in ~1 min
